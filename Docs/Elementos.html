Elementos comunes en los Lenguajes de Programación

Los lenguajes de programación se estructuran en 5 factores:

  1. Variables y Vectores

    Son espacios específicos de memoria que almacenan datos, en la mayoría de los lenguajes de programación se debe especificar el tipo de dato que va a ser almacenado como en Java en le que se escribe String si se quiere escribir una cadena de texto, pero en el caso de lenguajes como Python no es necesario ya que solo se necesita declarar la variable mas no el tipo de dato ya que el mismo puede ir cambiando acorde a la ejecución del programa. VECTORES: Mientras que una variable solo puede almacenar un valor (A excepción de las Strings ya que son un conjunto de caracteres) un vector puede contener un conjunto de valores almacenados de la misma manera, solo que en este caso representan espacios de memorias consecutivos.

    • char
    • int
    • float
    • string
    • boolean
  2. Condicionales

    Las sentencias condicionales permiten controlar el flujo del programa, ya que determinan la ejecucion de ciertas sentencias si es que se cumplen, por ejemplo, si un numero es mayor que otro, o si son iguales.

    • If
    • Else
    • Else If
  3. Bucles

    Los bucles son sentencias de codigo que permiten repetir cierto codigo un determinado numero de veces dependiendo si se cumple una condicion.

    • For
    • While

  4. Funciones / Metodos

    Es una manera de estructurar el codigo, ya que la funcionpuede ser llamada las veces que sean necesarias y pueden retornar valores para ser usados para llegar a una solucion.

  5. Sintaxis

    La manera en la que se escribe en un cierto lenguaje es conocido como sintaxis, varios lenguajes de programacion derivan su sintaxis del lenguaje C, la sintaxis define un conjunto de combinaciones para formar el funcionamiento del programa, dentro de la sintaxis de cada lenguaje existen dos tipos de semantica.

    1. Semantica Estatica

      Define las restricciones sobre la estructura de los textos validos que resultan complicados de expresar mediante formalismos, para los lenguajes compilados incluyen reglas que pueden ser verificadas al momento de compilar el programa, por ejemplo si una variable esta siendo usada y aun no a sido declarada.

    2. Semantica de Tipos

      Un sistema de tipos de datos define la manera en la que el lenguaje de programacion clasifica los valores y expresiones, y la manera en de interactuar entre si, por ejemplo, en la mayoria de lenguajes existen las Lagunas de Tipos, en la que un tipo de dato puede ser usado por otro, por ejemplo un int dentro de un float.

Implementacion

La implementacion de un lenguaje esta descrita en de dos maneras:

  1. Compilacion

    La compilacion es un proceso en el cual se traduce del lenguaje origen hacia otro, generando un equivalente que la maquina pueda entender, este proceso es realizado por un Compilador, que es el encargado de la traduccion, generalmente a lenguaje ensamblador. Lenguajes como C, C++, Lua, Ada, Haskell, Cobol son lenguajes compilados, el beneficio de la compilacion es un mejor rendimiento para el programa.

  2. Interpretacion

    Los lenguajes interpretados no son traducidos directamente a codigo maquina sino mas bien lo hacen linea por linea dependiendo si el programa necesita de dicha funcion, caso contrario no es tomada en cuenta, la ventaja de un lenguaje interpretado es la velocidad en el diseño ya que no tiene que estarse compilando cada ves que se desea ejecutar, mientras que su desventaja mas grande es la velocidad de ejecucion puesto que interpreta solo partes del codigo, luego las pasa a lenguaje maquina y sigue este proceso las veces necesarias.
    Lenguajes como Php, Python, Ruby son interpretados.

Java es un lenguaje que esta formado por ambas partes, es compilado, pero no directamente a lenguaje maquina sino que corre en su propia maquina virtual, lo que le hace, de cierta manera, interpretado.
